## Phần A : Kiểm tra đọc hiểu

## Câu A1:

- 10 input types khác nhau trong HTML:

1. type = "text" -> Ô nhập text, tự kiểm tra minlength, maxlength, pattern -> Dùng để nhập thông tin bất kỳ

2. type = "email" -> Ô nhập text, tự kiểm tra có @ → Dùng cho form đăng ký

3. type = "password" -> Ô nhập text và chuyển các text thành dấu chấm(ẩn ký tự), tự kiểm tra minlength, maxlength -> Dùng để nhập mật khẩu

4. type = "number" -> Ô nhập số có nút tăng giảm, tự kiểm tra min, max, step -> Cho người dùng chọn số lượng

5. type = "tel" -> Ô nhập bàn phím số, tự kiểm tra pattern -> Dùng để nhập số điện thoại

6. type = "date" -> Ô chọn ngày, tự kiểm tra min, max -> Dùng để nhập ngày

7. type = "time" -> Ô chọn thời gian, tự kiểm tra min, max -> Dùng để nhập thời gian

8. type = "color" -> Ô chọn màu, tự kiểm tra min, max -> Dùng để chọn 1 màu bất kì

9. type = "checkbox" -> Ô chọn có hoặc không, yêu cầu phải được tích -> Dùng để làm khảo sát

10. type = "radio" -> Ô chọn 1 trong nhiều lựa chọn, yêu cầu phải được chọn -> Dùng để làm khảo sát
